# MedYOLO11Qt 🏥

## 项目简介

**MedYOLO11Qt** 是一款基于 Qt 6 和 ONNXRuntime 的现代化医学图像 AI 分析软件，集成了 YOLO11 深度学习模型，专为医学影像分析和诊断而设计。

## ✨ 核心特性

### 🤖 AI 智能分析
- **双模式支持**：X 光异常检测 + MRI 图像分割
- **YOLO11 模型**：最新的目标检测和分割算法
- **智能缓存**：推理结果自动缓存，提升效率
- **模型保护**：内置加密工具保护知识产权

### 🏥 医学影像专业支持
- **DICOM 标准**：完整的 DICOM 3.0 格式支持
- **多格式兼容**：支持 BMP、PNG、JPG、TIFF 等常见格式
- **元数据展示**：专业的 DICOM 标签信息显示
- **批量处理**：支持文件夹批量分析

### 🎨 现代化用户界面
- **Qt 6 框架**：基于最新的 Qt 6.9.2 技术
- **直观操作**：简洁明了的用户界面设计
- **实时预览**：分析结果实时显示
- **多语言**：支持中英文切换

### ⚙️ 企业级功能
- **配置管理**：INI 格式集中配置管理
- **日志系统**：分级日志和自动轮转
- **错误处理**：完善的异常处理机制
- **一键部署**：专业的安装包生成

## 📋 系统要求

### 最低要求

- Visual C++ Redistributable 2022
- .NET Framework 4.7.2 或更高版本

## 🛠️ 技术架构

### 核心技术栈
| 技术 | 版本 | 用途 |
|---|---|---|
| **Qt** | 6.9.2 | 跨平台 GUI 框架 |
| **ONNXRuntime** | 1.22.1 | AI 推理引擎 |
| **GDCM** | 3.2.0 | DICOM 医学影像处理 |
| **C++** | 17 | 主要开发语言 |
| **CMake** | 3.18+ | 构建系统 |

### 架构设计
```
MedYOLO11Qt/
├── src/                    # 源代码（模块化设计）
├── models/                 # AI 模型文件
├── test/                   # 测试数据
├── build/                  # 构建输出
├── tools/                  # 工具程序
└── docs/                   # 项目文档
```

### 核心模块
- **MainWindow**：主界面和用户体验
- **InferenceEngine**：AI 推理引擎
- **AppConfig**：配置管理（单例模式）
- **ErrorHandler**：错误处理和日志
- **DicomUtils**：DICOM 文件处理
- **ImageView**：图像显示组件

## 🎯 核心功能演示

### X 光检测模式
1. **启动程序**：运行 `medapp.exe`
2. **选择模式**：选择 "X光检测" 模式
3. **加载图像**：打开 X 光图像文件
4. **自动分析**：程序自动进行 AI 分析
5. **查看结果**：检测结果显示在输出面板

1. **选择模式**：选择 "MRI分割" 模式
2. **加载 MRI**：打开 MRI 图像文件
3. **执行分割**：点击运行推理
4. **查看分割**：器官分割结果可视化显示

### 批量处理
1. **打开文件夹**：选择包含医学图像的文件夹
2. **自动扫描**：程序递归扫描支持的文件格式
3. **批量分析**：自动对所有图像进行 AI 分析
4. **结果汇总**：统一查看所有分析结果

## 🔧 配置管理

### 系统配置（config.ini）
```ini
[Paths]
QtPath=E:/tools/Qt/6.9.2/msvc2022_64
GDCMPath=E:/tools/GDCM-3.2.0-Windows-x86_64
ONNXRuntimePath=E:/tools/onnxruntime-win-x64-1.22.1

[Inference]
ConfidenceThreshold=0.25      # 检测置信度
IoUThreshold=0.45            # NMS 阈值

[Performance]
UseGPU=false                 # GPU 加速开关
BatchSize=1                  # 批处理大小

[UI]
Theme=Light                  # 界面主题
Language=Chinese             # 界面语言
```
